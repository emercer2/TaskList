# Summary
This app it a todo list app, not fully complete yet but pretty close. This is a React and Nextjs project.

Home: Choose a theme and navigate teh app
Team Progress (WIP): Page to manage a team's tasks. Current set up to a mock api, in the future I would like to connect it to my own so I can make new tasks and properly push them etc. Has mock data.
My Tasks (WIP): Manage your tasks. Currently uses local storage, eventually I would like to incorperate this into the above mentioned API. Has mock data.
About: It is an about page

# How to use
I have hosted this app using Vercel (https://vercel.com)

Run it yourself by:
1. Pulling down the repo
2. cd TaskListRoot
3. npm run dev

OR
Vist the hosted version on Vercel
